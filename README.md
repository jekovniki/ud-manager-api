## Description

API that uses the NestJS framework with Typescript. It uses swagger for documentation, zod for API validation, PostgreSQL for database, TypeORM for managing migrations, entities and everything related with database. UploadThing for uploading files, loggly for logs tracking and etc.

API can be found at:

```bash
/api-docs
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
